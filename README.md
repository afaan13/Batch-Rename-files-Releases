## ⚡ Patterns

| Pattern | What it does |
|---|---|
| 🔢 **Sequential** | Numbered renaming — `trip_001.jpg`, `trip_002.jpg`… |
| 🔍 **Regex** | Find & replace with capture groups — full control |
| 🔠 **Case** | `lowercase`, `UPPERCASE`, `Title Case`, `camelCase` |
| 💾 **Preserve** | Keep original name, add prefix/suffix |
| 📎 **Extension** | Rename only extensions — target specific types or all files |

Every pattern shows **live previews** before you commit. Filter, sort, and toggle individual files. Undo is always one tap away.
